# alx_fe_javascript

This repository contains frontend JavaScript projects completed as part of the **ALX Frontend Website Program** program.

---

## Project: Dynamic Quote Generator

A simple web application built with vanilla JavaScript that demonstrates:

- DOM manipulation
- Adding and displaying dynamic content
- Category-based filtering
- Web Storage (localStorage and sessionStorage)
- JSON import and export
- Simulated server synchronization
- Basic conflict resolution

---

## Technologies Used

- HTML
- JavaScript
- Web Storage API
- Fetch API
- JSON
- Mock API (JSONPlaceholder)

---

## How to Run

Open `dom-manipulation/index.html` in a web browser.
